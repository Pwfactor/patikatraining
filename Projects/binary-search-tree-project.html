Question :


[7, 5, 1, 8, 3, 6, 0, 9, 4, 2]
Yukarda verilen dizinin Binary-Search-Tree aşamalarını yazınız.

Answer:

[7, 5, 1, 8, 3, 6, 0, 9, 4, 2] Bu dizinin ilk elemanını (7) root olarak alalım cünkü sort edilmemis bir yapı var ve ilk defa binary search tree creation işlemi uygulanacaktir.
Buna göre root 7 dir. Rootu belirledikten sonra diğer rakamları tek tek sırasıyla sorgulayabiliriz.
